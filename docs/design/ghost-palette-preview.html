Proposed ghost teal tokens

Three values define the porcelain palette. Hues sit at 168°–173° to avoid the CartoDB water blue at 204°. Saturation stays low (17–21%) so the material reads as a study model, not as "real."

Proposed — porcelain substitution per element

Confidence is encoded as the proportion of the building rendered in faithful materials vs. porcelain teal. The teal palette stays constant; only the coverage varies. The eye still reads the building correctly because shape is preserved everywhere — only material varies per element (per brick, per window, per door, per roof).

90% confidence (≈10% porcelain)
60% confidence (≈40% porcelain)
30% confidence (≈70% porcelain)

↑ Each brick / window / door / roof is individually either faithful or porcelain. In production, the distribution is data-driven — the elements we have direct evidence for render faithfully, the elements we're inferring render in porcelain. This demo uses a deterministic pseudo-random scatter for visualization only.
